# mpvx
Mac app for mpv inspired by [grsyncx](https://github.com/username0x0a/grsyncx)

## Requirements

Install command-line interface mpv with [Homebrew](https://brew.sh).

```bash
$ brew install mpv
```

Make sure your mpv is installed in the correct path.

```bash
$ which mpv
/usr/local/bin/mpv
```
